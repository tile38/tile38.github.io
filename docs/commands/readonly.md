## Syntax

**READONLY yes|no**

## Description

Puts the server into readonly mode. While in this mode all commands that change data will fail such as SET and DEL.

## Examples

```tile38-cli
READONLY yes    # turns on readonly mode
READONLY no     # turns off readonly mode
```
